외장 모듈 사용하기
npm install nconf -> 해당 폴더에만 설치, 상위에 설치하면 편함

npm init -> package.json 생성
npm uninstall nconf -> 삭제 후,
npm install nconf --save -> 재설치, package.json에 dependencies 속성 추가

해당 package.json을 다른 PC로 옮긴 후 npm install하면 모든 패키지가 설치 됨.
